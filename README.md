## 🕹️ Controls Guide

| Key | Action |
|---|---|
| ← → | Move piece |
| ↑ | Rotate |
| ↓ | Soft drop |
| Space | Hard drop |
| C / LShift | Hold piece |
| P / Esc | Pause |

## 🏗️ Architecture Design Pattern

Game Entry
   ↓
Game Engine
   ↓
Playfield + Tetromino + Audio (SoundManager)

## ⚙️ Getting Started

### Prerequisites
* Visual Studio 2022  
* SFML 2.6.2  
* Windows 10/11  

### Build & Run
1. Open `.sln`
2. Set x64 Debug/Release
3. Press F5
